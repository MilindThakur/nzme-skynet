Skynet Setup
++++++++++++

Pre-Install
===========

Please make sure you have following installed:
    * Python 2.7
    * git
    * pip
    * virtualenv

Install Browsers and Drivers
============================

    Follow `drivers</drivers>`_ to install necessary browsers and Selenium drivers to run the scripts locally
